Resumo

Este documento diz respeito ao processo de gestão de um website utilizando servidores Node.js.

A partir dos ficheiros arq.xml e arq.xsl foram gerados os ficheiros html correspondentes: o index com o link localhost:1079/index e as páginas individuais com o link localhost:1079/arqelem/id, id esse que corresponde ao gerado para cada página.

Foi, então, criado um servidor para atender aos pedidos na porta 1079, sendo que este apresenta as páginas html requisitadas pelo cliente. Por exemplo, ao aceder a localhost:1079/arqelem/d1e3, o servidor lê o ficheiro d1e3.html da diretoria onde este se encontra e escreve-o (mostra ao cliente o seu conteúdo).
