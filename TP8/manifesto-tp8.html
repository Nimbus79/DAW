Resumo

Este documento diz respeito ao processo de criação de uma página web para o upload de ficheiros.

Inicialmente, foram criadas duas secções na página, uma para o upload e outra com a lista dos ficheiros submetidos.

De modo a facilitar a criação da lista referida anteriormente, sempre que é feito o upload de um ficheiro, as informações deste são guardadas num ficheiro json e, simultaneamente, estas são colocadas na lista através de uma função em jquery.

Por fim, o servidor foi criado com recurso à framework express do node.js, a atender pedidos na porta 7900.
