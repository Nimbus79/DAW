Resumo

Este documento diz respeito ao processo de criação de dois DTDs (Document Type Definition), um com base na página acerca do Processamento de matrículas ou inscrições no ensino ou em formação e outro com base no seguinte documento.

A partir do último DTD, foram também criados os respetivos ficheiros XML (Extensible Markup Language), XSL (Extensible Stylesheet Language) e HTML (HyperText Markup Language), sendo este último o presente documento.

Para obter os ficheiros referidos, foi, então, necessária a identificação de cada componente da página/documento, assim como a identificação dos seus respetivos atributos. Com isso, foi possível a elaboração do DTD, seguida da criação do XML, que teve como base este primeiro. Por fim, foi escrito o XSL com os comandos necessários para a criação da página HTML com a informação contida no XML, estando estes documentos na secção seguinte.
