Resumo

Este documento diz respeito ao processo de criação de páginas web, para a gestão de tarefas, usando a ferramenta pug para a criação dos templates.

Para se conseguir uma melhor gestão das tarefas por parte do utilizador, foram colocadas, na página principal, 4 opções: registar, consultar, apagar e concluir tarefas. Após o registo, remoção ou conclusão de uma tarefa, é mostrada uma página com as informações dessa mesma tarefa, em caso de sucesso.

De modo a facilitar a remoção de uma tarefa, antes de esta ser inserida na base de dados após o seu registo, é-lhe atribuída um identificador, único para cada tarefa. Assim, quando uma tarefa é apagada, basta procurá-la na base de dados pelo seu id e mudar a variável apagar para o valor "sim".

Caso o utilizador tente aceder a uma página que não esteja implementada ou, por algum motivo, ocorra qualquer falha, quer na leitura da BD quer no método pedido, é aberta uma página de erro.
