Resumo

Este documento diz respeito ao processo de criação de uma página web para o upload de ficheiros.

Inicialmente, foram criadas duas secções na página, uma para o upload e outra com a lista dos ficheiros submetidos. Caso esta lista esteja vazia, só fica visível, para o utilizador, a primeira secção e, sempre que é feito um novo upload, a lista é atualizada com os detalhes do ficheiro submetido.

De modo a facilitar a criação da lista referida anteriormente, sempre que é feito o upload de um ficheiro, as informações deste são guardadas num ficheiro json.

Por fim, o servidor foi criado com recurso à framework express do node.js, a atender pedidos na porta 7900.
