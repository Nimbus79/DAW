Resumo

O teste é composto por dois exercícios: Compositores musicais e CLAV.

No primeiro exercício foi convertido o ficheiro XML num ficheiro JSON e, de seguida, os dados desse ficheiro JSON foram importados para o mongo. Para aceder às funcionalidades pedidas, como listar os compositores, basta entrar no link http://localhost:3001/. Para testar o programa foi utilizado o Postman, sendo também necessário ter o mongo a correr.

Na primeira parte do segundo exercício era pedido a realização de três queries, embora só tenha sido conseguida, com sucesso, a segunda. Na segunda parte do exercício foi criada uma interface com as informações das classes da CLAV. Em cada página da classe encontrava-se um link para regressar à página princial (http://localhost:3002/), assim como toda a informação relevante dessa mesma classe. Para conseguir realizar este exercício foi necessário obter os dados, através do axios.
